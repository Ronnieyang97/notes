wget 从某处下载
sudo 用管理员身份运行
sudo apt install 最简单的安装
ifconfig 查看网络情况
uname 查看当前系统版本
uptime 查看系统负载情况
free 查看内存使用
who 查看登录的账户
last 查看登录记录
history 查看历史命令
.开头的是隐藏文件，一般是系统文件
pwd 显示当前工作目录
ls 查看当前目录下的文件  -a 查看全部文件，包括隐藏  -d 仅查看目录本身  -h 易读的文件容量  -l 显示文件的详细信息
cat 查看较短的纯文本文件  -n 显示行号  -b显示不包括空行的行号  -A 显示出不可见的符号，如空格、tab等
more 查看较长的纯文本文件 -数字 预先显示的行数
head 查看纯文本文件的前n行 -n 数字
tail 查看文本的后n行 -n 数字
od 查看特殊格式的我呢见 -t a 默认字符  -t c ASCII字符  -t o（d/x/f） 八进制/十进制/十六进制/浮点数
cat 文件 | tr [原始字符] [目标字符]  替换目标字符
wc -l/-w/-c 文件  统计目标文件中的行数，字数，字节数
cut -d(分隔符) -f（number）  通过列来提取想要的信息，number为第n列
diff [参数] 文件  比较文本文件的差异  -b 忽略空格引起的差异  -B 忽略空行引起的差异  
touch [选项] [文件]
	touch test 可以创建空白文档
	touch -d "2 days ago" test 将访问和修改时间改到了两天前
	-m 修改更改时间  -a 修改访问时间  -d 同时修改两者 -t 修改成[YYMMDDhhmm]
mkdir [选项] 目录
cp [选项] 源文件 目标文件  -p 保留原始文件属性  -r 递归持续复制（用于目录）  -i 若目标文件存在则询问是否覆盖
mv [选项] 文件名 [目标路径|目标文件名]  移动文件
rm [选项] 文件 删除文件
dd [参数] 用于指定大小的拷贝文件或指定转换文件  if=输入的文件名称  of=输出的文件名  bs=设置每个块的大小  count=设置拷贝块的个数
useradd [选项] 用户名 创建新的用户  -d 指定用户的家目录  -D展示默认值  -e 账号截至日期 YYYY-MM-DD  -g 指定一个已存在的初始用户组  -s 指定默认的shell  -u 指定用户的UID
passwd [选项] [用户名]  修改用户的密码  -l 锁定账户  -u 解除锁定  -d 使账号无密码  -e 强制用户更改密码
userdel [选项] 用户名 删除用户所有表格  -f 强制删除用户  -r 同时删除用户
usermod [选项] 用户名  修改用户的属性  -c 填写备注  -d -m 重新指定家目录并迁移旧数据  -e 账户到期时间“YYYY-MM-DD”  -g 更改用户组  -L 锁定用户禁止其登录系统  -U 解锁用户  -u 修改用户的UID
groupadd [选项] 群组名  创建群组
tar [选项] [文件] 对于文件打包解压 -c 创建压缩文件 -x 解开压缩文件  -t 查看压缩包内的文件  -v 显示压缩或解压过程  -p 保留原始的权限与属性  -P使用绝对路径来压缩
grep [选项] [文件]  用于对文本进行搜索  “grep 关键词 文本文件”  -b将可执行文件当作文本搜索  -c 仅显示找到的次数  -i 忽略大小写  -n 显示行号
find [查找路径] [寻找条件] [操作]  ~：家目录 .当前目录 /根目录  -name 匹配名称
